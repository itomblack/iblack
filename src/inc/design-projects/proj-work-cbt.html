The problem

How can you educated busy staff?

The Process

A workshop was run by pebble {code}, to understand the training requirements of the client's employees. The main aim of this training course being to promote the benefits of the new Adaptive Delivery Framework.

An empathy mapping exercise around staff personas identified the users' key attributes and expectations.

From this analysis the following criteria were constructed:

  • Modular
  • Adaptable
  • Concise
  • Mobile
  • Mixed Media
  • Personal

A four-week build schedule was proposed.Wireframes for a set of training module were produced. Each module would consist of a three-minute animation, some reading and a short quiz. Users could choose to learn through video or text, but had to finish at least one before attempting to pass the quiz.

It was built as a static site, before being into the client's SCORM system. It is browser compatible down to IE9.
